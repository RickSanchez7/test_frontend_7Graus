# 7Graus Challenge

An app to fetch users names and images.

### Stack:

- React;
- Typescript;
- CSS;
- Jest and React-Test-Library;
- Eslint + Prettier;
- Default webpack configuration used by create-react-app;

### Commands:

##### Initialize Project Development:

- npm start

##### Run tests:

- npm test

##### Build Production Bundle:

- npm run build
